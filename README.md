## Tested On

- [x]  Windows
- [x]  macOS
- [x]  Linux (Pop!_OS 20.04 LTS)

## Installation

Install my project with git and cargo (cargo and rust can both be installed using rustup)
```bash
    git clone https://github.com/shaleen111/snake-rust.git
    cd snake
    cargo build --release
```

## Usage/Examples

```
  cargo run
```
